# Breathing AI

A computer vision project that uses your webcam to detect and classify breathing phases in real time.

The camera tracks face and shoulder movement using body landmark detection. That movement signal is recorded, labeled, and used to train a neural network that can recognize four breathing states: **inhale**, **exhale**, **hold in**, and **hold out**.

---

## How It Works

```
Webcam -> MediaPipe detects body -> face + shoulder landmarks recorded
       -> scipy smooths the signal
       -> data normalized + sliced into 60-frame windows (~2 seconds)
       -> Conv1D + LSTM neural network trained on those windows
       -> live predictions from webcam
```

## CSV Data Format

Each session file contains one row per camera frame, with x and y position for each landmark:

| Column group | Landmarks |
|---|---|
| `nose_x`, `nose_y` | Nose tip |
| `left_eye_inner_x/y` ... `right_eye_outer_x/y` | Both eyes (inner, center, outer) |
| `left_ear_x/y`, `right_ear_x/y` | Both ears |
| `mouth_left_x/y`, `mouth_right_x/y` | Mouth corners |
| `left_shoulder_x/y`, `right_shoulder_x/y` | Both shoulders |
| `shoulder_width` | Distance between shoulders (normalization helper) |
| `timestamp` | Seconds elapsed since recording started |
| `label` | `inhale` / `exhale` / `hold_in` / `hold_out` |

Elbows and wrists are excluded — they move independently when using a mouse.

---

## Processed Dataset Format

`data/processed/dataset.npz` is a compressed numpy file containing two arrays:

| Array | Shape | Description |
|---|---|---|
| `X` | `(9098, 60, 26)` | 9098 training windows, each 60 frames x 26 features |
| `y` | `(9098,)` | Label for each window: 0=inhale, 1=exhale, 2=hold_in, 3=hold_out |

Generated by running `process_data.py`. Re-run it any time you add new sessions.

---

## Model Architecture

- **Input:** 60-frame sliding window, 26 features per frame
- **Conv1D(26 -> 32, kernel=5)** — detects local patterns (rising/falling signal)
- **MaxPool1D** — downsamples from 60 to 30 time steps
- **LSTM(32 -> 64)** — learns patterns across the sequence over time
- **Dropout(0.3)** — prevents overfitting
- **Dense(64 -> 4)** — outputs one score per class
- **Softmax** — converts scores to probabilities

Total parameters: 29,540

## Installation

**1. Clone or download this project**

**2. Install all dependencies using Python 3.12 specifically**

```bash
py -3.12 -m pip install opencv-python mediapipe scipy numpy pandas matplotlib torch jupyter
```

**3. Verify everything installed correctly**

```bash
py -3.12 -c "import cv2, mediapipe, torch; print('All good! PyTorch version:', torch.__version__)"
```

**4. That's it.** The MediaPipe pose model (~3MB) is downloaded automatically on first run.

> **Note:** Always use `py -3.12` to run scripts in this project, not just `python` or `py`, to make sure you're using the correct Python version.

## Recording Data

```bash
py -3.12 scripts/collect_data.py
```

- A window opens showing your webcam with a skeleton overlay
- Press a key when each breathing phase starts — the label stays until you press the next key
- **W** = inhale &nbsp; **S** = exhale &nbsp; **E** = hold_in &nbsp; **D** = hold_out
- Press **Q** to stop — data is saved automatically to `data/session_XXX.csv`

**Tips:**
- Fill the frame with your upper body (shoulders visible)
- Light source in front of you, plain wall behind
- Aim for 10-15 full breathing cycles per session (~2-3 minutes)

---

## Processing Data

Run this after recording new sessions to regenerate the training dataset:

```bash
py -3.12 scripts/process_data.py
```

Smooths all signals, normalizes values, slices into 60-frame windows, saves `data/processed/dataset.npz`.

---

## Training the Model

```bash
py -3.12 scripts/train_model.py
```

Trains the Conv1D + LSTM model and saves the best weights to `models/breathing_model.pt`.
Key settings in the script: `EPOCHS`, `BATCH_SIZE`, `LR` (learning rate).
